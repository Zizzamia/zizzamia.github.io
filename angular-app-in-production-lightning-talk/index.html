Disabling Debug Data

Disabling Debug Data

talkApp.config(['$compileProvider', function ($compileProvider) {
  $compileProvider.debugInfoEnabled(false);
}]);

Debug an application with debugInfoEnabled False

angular.reloadWithDebugInfo();

Tools like Protractor and Batarang need this information to run, but you can disable this in production for a significant performance boost.

How big is this performance boost ?

Run Benchpress on a largetable with and without debug-info

protractor protractor-js.conf.js --benchmark --spec largetable
protractor protractor-js.conf.js --benchmark --spec largetable-no-info
